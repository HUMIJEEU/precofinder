# Deploy do Frontend (PWA) no Netlify — Windows

1) Repositório GitHub: https://github.com/HUMIJEEU/precofinder (pasta `/frontend`).
2) Acede a https://app.netlify.com → **Add new site** → **Import from GitHub** → escolhe o repositório.
3) Como é um site estático, não precisas de build: aponta a pasta de publicação para **/frontend**.
4) Depois do deploy, abre o site e define a API do backend (Render) clicando em **Definir API** e colocando o URL, ex.:
   `https://melhorprecofinder.onrender.com`
5) Podes ligar um domínio temporário do Netlify (ex.: `precopreinder.netlify.app`) e, mais tarde, o teu domínio .com.
6) Em telemóvel, usa **Adicionar ao Ecrã Inicial** (PWA).
